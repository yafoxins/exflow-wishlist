## üíª Local Development

<details>
<summary><b>üêç Backend Setup</b></summary>

<br>

```bash
# Navigate to directory
cd backend

# Create virtual environment
python -m venv venv

# Activate
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# Install dependencies
pip install -r requirements.txt

# Setup environment
cp .env.template .env
# Edit .env

# Apply migrations
alembic upgrade head

# Start server
uvicorn app.main:app --reload
```

‚úÖ **Backend available at:** http://localhost:8000

</details>

<details>
<summary><b>‚öõÔ∏è Frontend Setup</b></summary>

<br>

```bash

# Install dependencies
npm install --legacy-peer-deps

# Setup environment
cp .env.example .env
# Set REACT_APP_API_URL=http://localhost:8000/api/v1

# Start server
npm start
```

‚úÖ **Frontend available at:** http://localhost:3000

</details>

## ‚öôÔ∏è Configuration

<details>
<summary><b>üîê Environment Variables</b></summary>

<br>

### Root `.env`

```env
POSTGRES_USER=exflow_user
POSTGRES_PASSWORD=your_secure_password_here
POSTGRES_DB=exflow_db
```

### Backend `.env`

```env
# Database
DATABASE_URL=postgresql+asyncpg://user:password@localhost/exflow_db

# Redis
REDIS_URL=redis://localhost:6379/0

# Security ‚ö†Ô∏è CHANGE THIS!
SECRET_KEY=your-super-secret-key-change-this
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=1440
REFRESH_TOKEN_EXPIRE_DAYS=30

# OAuth (optional)
YANDEX_CLIENT_ID=your_client_id
YANDEX_CLIENT_SECRET=your_client_secret

# Telegram (optional)
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHI...
TELEGRAM_WEBHOOK_URL=https://your-domain.com/api/v1/telegram/webhook

# Email
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASSWORD=your-app-password
```

### Frontend `.env`

```env
# For Docker
REACT_APP_API_URL=/api/v1

# For local development
REACT_APP_API_URL=http://localhost:8000/api/v1
```
